# Tổng hợp kiến thức
## Git Command
- `git commit --amend` -> thay đổi message của commit gần nhất  
- `git restore --staged <file>` -> bỏ file khỏi staged và đưa về working directory  
- `git reset --soft HEAD~1` -> undo commit cuối, giữ thay đổi trong stage  
- `git reset HEAD~1` -> undo commit cuối, giữ thay đổi trong working directory  
- `git branch <name>` -> tạo nhánh mới  
- `git checkout <name>` -> chuyển sang nhánh `<name>`  
- `git branch -D <name>` -> xoá nhánh  

## JavaScript condition
- câu điều kiện `if` -> check điều kiện và chạy câu lệnh nếu thoả
- vòng lặp `for` -> chạy vòng lặp khi điều kiện vẫn được thoả
